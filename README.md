# MiniSheet

## One-Line Pitch
Lightweight spreadsheet for quick calculations, no Excel or Google Sheets overhead.

## Features
- 10x10 editable grid with row/column labels
- Formula support with cell references
- Automatic recalculation on dependency change
- Cell selection shows raw formula or value

## Getting Started
```bash
npm install
npm run dev
npm run build
```
Deploy to your browser and start calculating!
